Welcome to the GPS API!

This API is based on REST (Representational State Transfer) architecture. The main task of this api is responding to HTTP requests initiated by applications
in order to receive or pass useful data. The required data storage format is Javascript Object Notation (JSON) format.

You can use the URLs listed below to access the required resources:

  • https://gpsproject.pl/coordinate

  • https://gpsproject.pl/coordinates

  • https://gpsproject.pl/settings
